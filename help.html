API Calls

There are many system functions which can be triggered via an http request. Lights, Cameras and Video switcher can be controlled this way.

Camera Cues

The general form of the camera cue url is: 
 
http://<hostname>:<port>/cq?cmd=___&parm=____&num=___
cmd parm num Example
power on|off camera # http://127.0.0.1/cq?cmd=power&parm=on&num=1
tally on|off camera # http://127.0.0.1/cq?cmd=tally&parm=on&num=1
move left|right|up|down|in|out|stop|zoomstop camera # http://127.0.0.1/cq?cmd=move&parm=left&num=1

Light Cues

The general form of the light cue url is: 
 
http://<hostname>:<port>/lq?cmd=___&parm=____
cmd parm Example
gotocue xxx.y http://127.0.0.1:1880:/lq?cmd=gotocue&parm=123.4
pause
go
goback

Video Cues

http://<hostname>:<port>/vq?cmd=___&parm=____
